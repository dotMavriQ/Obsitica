## Introduction
Obsitica is a sophisticated script developed initially in PowerShell, with a future roadmap for Python integration. It is designed to automate the organization of your Habits, Dailies, and achieved To-Do's into Obsidian. This tool seamlessly places these entries into your Obsidian JOURNAL folder, enhancing your productivity and tracking efficiency.

## Features
- **Habit Tracking**: Automatically places your tracked habits into Obsidian.
- **Daily Task Management**: Integrates your daily tasks into the JOURNAL folder.
- **To-Do List Integration**: (In Development) Aims to add achieved To-Do's into your daily entries.

## Setup Requirements
Before using Obsitica, ensure the following setup in your Obsidian workspace:
- A folder named `JOURNAL`.
- Use of the default date format (YYYY-MM-DD) for the calendar plugin.
- An entry created for the current day in the `JOURNAL` folder.

## Installation and Usage
(Currently available in PowerShell. Python version in planning.)
1. Clone or download the Obsitica script.
2. Place it in a suitable directory within your Obsidian workspace.
3. Execute the script in PowerShell, ensuring you have the necessary permissions.
